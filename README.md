# Smart Expense Tracker with Insights

## Project Description
Smart Expense Tracker is a Python-based application that helps users record, categorize, and analyze their daily expenses.  
It provides monthly summaries, category-wise expense breakdowns, pie chart visualization, highest spending category detection, and smart suggestions to help users manage their spending better.

## Problem Statement
Many individuals struggle to track daily expenses and understand their spending patterns.  
This project provides a simple and user-friendly solution to log expenses and generate useful insights.

## Objectives
- Record daily expenses with:
  - Date
  - Category
  - Amount
  - Description
- Categorize expenses (Food, Travel, Bills, Shopping, Others)
- Generate monthly summaries and reports
- Visualize category-wise spending using pie charts
- Detect highest spending category
- Provide smart suggestions for reducing expenses

## Features
- Add new expenses
- View all recorded expenses
- Monthly expense summary
- Category-wise breakdown
- Highest spending category detection
- Pie chart visualization using matplotlib
- Smart insights and suggestions
- Monthly expense report

## Technologies Used
- Python
- Tkinter (GUI)
- CSV File Handling
- Matplotlib
